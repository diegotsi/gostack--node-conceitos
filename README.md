# GoStack - Challenge 01 - Node JS

This project provides a simple Repositories` CRUD

### Running

`yarn`
`yarn dev`
The project should be available on `localhost:3333/repositories`

### Available routes

```
    [GET] /repositories
    [POST] /repositories
    [PUT] /repositories/:id
    [DELETE] /repositories/:id
    [POST] /repositories/:id/like
```

### Running tests

`yarn test`
